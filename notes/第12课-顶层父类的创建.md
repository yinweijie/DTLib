# 重载new和delete操作符

不同编译器`new`失败后的动作不一定完全一样，有的会抛出异常，有的会返回空指针，因此为了打造通用平台库，最好自定义`new`和`delete`操作符，其接口为：

```cpp
class Object
{
public:
    void* operator new(unsigned int size) noexcept;
    void operator delete(void* p);
    void* operator new[](unsigned int size) noexcept;
    void operator delete[](void* p);
    virtual ~Object() = 0;
};
```
